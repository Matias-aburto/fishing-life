# Iconos de objetos (inventario)

Mismo estilo que los peces: **PNG**, fondo transparente, **32×32 px** para los slots del HUD.

| Archivo | Objeto | Slot en juego |
|---------|--------|---------------|
| `rod.png` | Caña de pesca | Caña |
| `bait.png` | Carnada | Carnada |

Nombres fijos (no cambiar): el código los buscará así.

## Import en Godot

Filter: **Off**, Mipmaps: **Off**.

## Ruta en código (futuro)

`res://assets/sprites/items/icon/rod.png`  
`res://assets/sprites/items/icon/bait.png`
