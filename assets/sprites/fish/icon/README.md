# Iconos de peces (inventario / UI)

Versiones pequeñas para el HUD y la tienda. Mismo `species_id` que la carpeta padre.

| Archivo | Tamaño sugerido |
|---------|-----------------|
| `pejerrey.png` | 24×24 px |
| `perca.png` | 24×24 px |
| `trucha_arcoiris.png` | 24×24 px |
| `trucha_fario.png` | 24×24 px |
| `carpa.png` | 24×24 o 32×32 px |

## Ruta en código

`res://assets/sprites/fish/icon/{species_id}.png` — usado por `FishIcons` en el inventario.

Si se ven borrosos en Godot: selecciona los PNG → Import → **Filter: Off**.
